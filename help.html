Sardines 101: A quick crash course

Sardines is a tool to make large volume community moderation jobs a little easier. The basic gist of it is really simple: you probably have lots of canned responses and would like to sift through them and copy them elsewhere more easily!

To begin, you must first load a JSON file. The schema that the tool expects can be found here, however a simpler to use visual editor is coming Soon™! If you're reading this currently, it's likely that I'll have provided you with a pre-made JSON file, so load that up and modify it as needed!

Once you have a JSON file loaded, you can simply click on a card to copy its contents to your clipboard! Right click it to favorite it - favorited cards will always be displayed before others. Right click it again to un-favorite it! That's... really it, for now! For posterity, the search box looks through both the title and description of each card and is fairly forgiving!

If you have any recommendations for or questions about this tool please don't hesitate to reach out! I'd like to improve it to better serve everyone's needs where possible.
